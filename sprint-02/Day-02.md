# Day 2 - Sprint Week 2

Today I focused on refining the chatbot. I fixed many issues that would arise with the bot, such as reading from the JSON while it was being written to causing an error
with the data the bot would receive. I also increased the range of questions the bot may be asked, as well as refining the responses the bot would give back, such as adding
more variety, human-like text, and advice based on the damage, speed or braking distance of the car. 

I also fixed a substantial amount of errors that would occur when the bot did not understand a question it had been asked. I solved this by "training" the model 
to understand the various ways or nuances that the same question may be asked,in order to improve the accuracy of which it would recognise the question. I also made the
chatbot window appear alongside TORCs when the torcs_jm_par.py file is ran, which allows for an ease-of-access way to write to the chatbot without having to type the
chatbot python file name in the command prompt to run it.
