# Day 1 - Sprint Week 2

Today I began work on implementing an IBM Chatbot known as "Watsonx Assistant", which we learned from the Build Your First Chatbot course in IBM Skillsbuild, 
into our project, to begin the development of a race engineer chatbot who could assist questions the user may ask about the car, such as 
"What is my speed?", "What was my lap time that lap?", "Should I pit?", or "How damaged is the car?".

To start this, I signed up for IBM Cloud, and ran a free version of the Watsonx Assistant known as the Lite version. I then began to set up actions for the assistant,
which means that the Chatbot will begin to recognise questions that it may be asked. After this, I began implementing it into a python file by creating a session with
the watson assistant by passing through the API Key and using the IBM_Watson library in python.

The Assistant can recognise variables when passed through in the code, which made passing through our current variables relatively seamless with only a few errors that
I plan to fix within the next day or at most 2 days.
