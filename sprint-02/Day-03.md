# Day 3 - Sprint Week 2

Today was mostly finalising the chatbot and improving existing features. I managed to allow the bot to read the post-lap analysis stats back to the user for a more informed
and life-like chatbot. I did this by passing through the CSV data into the file and then parsing the data into variables that would then be passed to the chatbot.
The chatbot would then read what lap the user was on in order to make sure a new lap had passed, so that it would read the correct data.

I also began helping Ryan to implement some graphs of the car that will pop up after the car has completed x amount of laps in order to further analyse and read the data
from the car. We are using python libraries such as matplotlib for the graphs and more.
