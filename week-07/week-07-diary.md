# Week 7 - LSEP

In week 7, we talked about the Legal, Social, Ethical and Professional issues that could come up in our project. When we looked at UI/UX ethics and LSEP, we realised
that a lot of these issues don’t really apply to our project. This is because we don’t interact with users, we don’t have a user interface, and we don’t store any 
data. Even so, there are still a few things we need to be careful about.

Legal Issues:
We can change the car liveries, as well as some garage doors and signs. But using a brand logo or someone else’s artwork without permission, or without it being 
fair use, could break laws like the Trade Marks Act 1994 and the Copyright, Designs and Patents Act 1998. Because of this, we will only use artwork that we’ve made 
ourselves or that is available under Creative Commons. Also, if we use or edit any code we find online, we need to give proper credit, as it could also be protected 
by similar laws.

Social/Ethical:
We need to create a walkthrough or guide for our racebot. When making this, we have to make sure it’s clear, easy to follow, and accessible. 
It should be understandable for people with different skill levels, and any documents we create should be easy to read, with clear fonts, good colours, and 
appropriate text size.
